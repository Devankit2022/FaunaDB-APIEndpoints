API EndPoints :

HTTP GET method

  • /customer - get complete details of all Customers.
  • /manager - get complete details of Manager.
  • /order - get complete details of all Orders.
  • /product - get complete details of all Products.
  • /store - get complete etails of all Stores.

HTTP POST method (use postman)

  • /customer - create a document in Customer collection using JSON data.

HTTP PUT method (use postman)

  • /customer - update the data of a document in Customer collection, using JSON data , you need to pass the document id in the url.

HTTP DELETE method (use postman)

  • /customer - delete a document in Customer collection, you need to pass the document id in the url.
